# Проект: "Топ-чарты Spotify (2015-2025)"

Этот проект посвящён исследованию глобальных музыкальных трендов за десятилетие на основе данных Spotify Top Charts (2015–2025). Цель работы - выявить ключевые паттерны популярности треков, определить изменения в музыкальных предпочтениях слушателей и продемонстрировать практические навыки работы с данными в Python и Yandex DataLens.

### Данные

В основе проекта лежит датасет Spotify Top Charts 2015–2025 (примерно 85 тыс. строк), содержащий информацию о треках, артистах, стримах, позициях в чартах и связанных метаданных. Формат — CSV. Данные загружены в Python с помощью pandas.

### Этапы анализа

1. Анализ данных в Python (pandas, numpy)
В Google Colab выполнены основные этапы:
A. Первичный обзор
B. Очистка и обработка данных
C. Статистический анализ
D. Визуализации (5+ графиков)
E. Группировки и агрегации

2. Aналитика в Yandex DataLens
Для демонстрации навыков построения интерактивной визуальной аналитики датасет загружен в DataLens.
Собран интерактивный дашборд с фильтрами, селекторами и управлением отображением.

3. Публикация на GitHub
Проект опубликован в публичном репозитории и включает:
Jupyter Notebook с полным анализом данных;
исходный CSV-датасет;
файл README.md;

### Итоги

Работа демонстрирует полный цикл анализа данных: от загрузки и очистки до визуализации, BI-аналитики и оформления результатов. Проект может служить основой для углублённого исследования музыкальных трендов и расширения навыков работы с данными и визуализацией.
