# Проект: "Топ-чарты Spotify (2015-2025)"

Этот проект включает всесторонний анализ данных о музыке Spotify с 2015 по 2025 год. Цель состоит в том, чтобы выявить тенденции в характеристиках музыки (популярность, танцевальность, энергичность, количество прослушиваний) на протяжении многих лет, понять распределение различных признаков и подготовить данные для визуализации и интерактивного исследования.
